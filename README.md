# Spark Streaming Cookbook

This repository is a collaborative collection of scenarios, solutions, and best practices for working with Apache Spark Streaming and Structured Streaming.

It serves as a reference guide for data engineers tackling real-time data processing challenges. Each scenario includes a clear problem description, architecture outline, tooling used, code snippets (where applicable), and recommendations based on practical experience.

The goal is to provide reusable patterns and guidance for building reliable, scalable streaming data pipelines.

## Structure

- `pyspark_park_streaming.ipynb/` – Real-world pyspark streaming scenarios with solutions
- `structured-streaming.md/` – Real-world pyspark structured streaming scenarios with solutions
- `README.md` – Overview of the project and its purpose
- `template.md` – Overview of the project guide and scope for team collaboration

## Usage

Browse the `scenarios` folder to explore various streaming use cases. Each markdown file follows a consistent format for quick understanding and reuse.
